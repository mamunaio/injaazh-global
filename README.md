## 0x04. QUICK START (DEVELOPMENT)

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

```bash
# 1. Clone the legacy
git clone https://github.com/mamunaio/injaazh-global.git

# 2. Enter the machine
cd injaazh-global

# 3. Install dependencies
npm install

# 4. Fire up the engine
npm run dev
```

The engine will be live at `http://localhost:3000`.

## 0x05. PROJECT STRUCTURE
We have moved away from the redundant `src` directory for a direct, high-performance root-level structure:

```text
├── app/          # Kinetic layouts & server-side routes
├── components/   # Surgical UI components
├── public/       # Cinematic assets & branding
├── lib/          # Utilities & logic
├── tsconfig.json # TypeScript architecture
└── package.json  # Engine configuration
```
